# Sawyer AI - Unity Development Assistant

An AI-powered Unity development assistant that provides intelligent code generation, project management, and development workflow automation.

## Features

- **AI Code Generation**: Generate Unity scripts, components, and systems using natural language
- **Intelligent Project Analysis**: Understand your project structure and provide contextual assistance
- **Automated Workflow**: Streamline common Unity development tasks
- **Real-time Collaboration**: Connect with Python backend for advanced AI capabilities
- **Context-Aware Suggestions**: Get relevant code suggestions based on your current work

## Installation

### Via Unity Package Manager
1. Open Unity Package Manager (`Window > Package Manager`)
2. Click the `+` button and select `Add package from git URL`
3. Enter: `https://github.com/suhail-Rahman/Test_UPM.git`

### Via Package Manager UI
1. Open Unity Package Manager
2. Select `My Registries` or `In Project`
3. Find and install `Sawyer AI`

## Quick Start

1. After installation, access Sawyer AI via `Window > Sawyer AI`
2. Configure your AI settings in the Settings panel
3. Start using natural language to generate code and manage your project

## Requirements

- Unity 2022.3 LTS or newer
- .NET Framework 4.7.1 or .NET Standard 2.1
- Python 3.8+ (for advanced AI features)
